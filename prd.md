# 私域 Agent MVP 版本产品需求文档(PRD)

## 📋 产品概述

### 产品名称

私域运营智能助手 MVP 演示版 (Private Domain Agent Demo)

### 产品定位

通过对话式 AI 交互演示完整的私域运营流程创建体验，展示如何将复杂的 SOP 创建、用户旅程设计简化为自然语言对话。

### MVP 目标

- **演示完整流程**：展示四步对话流程的完整用户体验 ✅
- **验证交互设计**：测试对话式界面和 Artifact 编辑的用户接受度 ✅
- **技术可行性验证**：验证前端架构和组件设计的合理性 ✅
- **快速迭代基础**：为后续功能扩展提供稳定的技术基础 ✅

### 核心简化原则

- **前端交互优先**：重点实现用户界面和交互体验 ✅
- **模拟数据驱动**：所有 AI 处理结果用预设的模拟数据 ✅
- **最小可行实现**：专注核心流程，去除复杂的后端逻辑 ✅
- **演示效果导向**：确保每个步骤都有清晰的视觉反馈 ✅

## 🎯 MVP 功能范围

### ✅ 已实现功能

#### 1. 完整的四步对话流程

- **步骤 1：场景选择**

  - 4 个预设场景（培育、邀约、发售、交付）✅
  - 场景卡片展示，支持单选 ✅
  - 每个场景包含详细的描述和适用场景 ✅

- **步骤 2：业务信息收集**

  - 支持自然语言输入业务描述 ✅
  - 文件上传功能（支持 PDF、Word、文本等）⚠️ 进行中
  - 智能信息提取和结构化展示 ✅
  - 支持人工编辑和确认 ✅
  - 字段包含：行业、产品、目标受众、优势、痛点等 ✅

- **步骤 3：内容序列生成**

  - 基于场景的智能内容生成 ✅
  - 支持批量生成（3-5 条）✅
  - 支持继续生成功能 ✅
  - 内容覆盖天数自动计算 ✅
  - 每条内容包含：
    - 标题和描述 ✅
    - 发送时间设置 ✅
    - 内容预览和编辑 ✅
    - 字数统计 ✅

- **步骤 4：用户分层配置**
  - 两种分层方式：
    1. 新客户 SOP ✅
       - 自动为新加企微好友创建跟进流程 ✅
       - 自动生成任务配置 ✅
    2. 特定条件用户 ✅
       - 智能条件识别系统 ✅
       - 自动判断是否可系统化执行 ✅
       - 配置要求智能提示 ✅
       - 标签名称智能推荐 ✅
       - 支持手动/自动标签切换 ✅

#### 2. 智能条件识别系统

- **可自动化条件类型**：

  - 行为类：查看、浏览、点击、购买等 ✅
  - 互动类：发消息、咨询、回复等 ✅
  - 时间类：加入时间、注册时间等 ✅

- **判断规则**：

  - 必须包含具体数值条件 ✅
  - 必须包含可自动化的行为/状态 ✅
  - 支持时间范围限定 ✅

- **智能提示**：
  - 自动化条件：提示所需系统配置 ✅

  - 智能生成标签名称 ✅

## 部署说明

项目已实现静态部署支持，可通过以下方式部署：

1. 本地开发：

```bash
npm install
npm run dev
```

2. 生产部署：

```bash
npm run build
npx serve out
```

访问 [http://localhost:3000](http://localhost:3000) 查看应用。
